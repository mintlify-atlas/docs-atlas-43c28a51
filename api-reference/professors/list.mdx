---
title: 'List professors'
description: 'Retrieve all professors belonging to the authenticated user.'
---

`GET /api/professors`

Returns a list of all professors owned by the authenticated user.

## Authentication

This endpoint requires a valid Bearer token in the `Authorization` header.

## Response

Returns an array of professor objects.

<ResponseField name="id" type="string" required>
  Unique identifier for the professor (UUID).
</ResponseField>

<ResponseField name="fullName" type="string" required>
  The full name of the professor.
</ResponseField>

<ResponseField name="userId" type="string">
  The ID of the user who owns this professor. May be `null` for system-level records.
</ResponseField>

## Error codes

| Status | Description |
| ------ | ----------- |
| `401 Unauthorized` | Missing or invalid Bearer token. |

<RequestExample>

```bash cURL
curl --request GET \
  --url https://api.scheduly.app/api/professors \
  --header 'Authorization: Bearer <token>'
```

</RequestExample>
