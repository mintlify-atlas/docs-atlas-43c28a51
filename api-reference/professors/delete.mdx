---
title: 'Delete professor'
description: 'Permanently delete a professor by their ID.'
---

`DELETE /api/professors/:id`

Permanently deletes a professor. The professor must belong to the authenticated user. This action cannot be undone.

## Authentication

This endpoint requires a valid Bearer token in the `Authorization` header.

## Request

### Path parameters

<ParamField path="id" type="string" required>
  The UUID of the professor to delete.
</ParamField>

## Response

Returns `204 No Content` on success with an empty response body.

## Error codes

| Status | Description |
| ------ | ----------- |
| `401 Unauthorized` | Missing or invalid Bearer token. |
| `404 Not Found` | No professor with the given ID exists for the authenticated user. |

<RequestExample>

```bash cURL
curl --request DELETE \
  --url https://api.scheduly.app/api/professors/c3d4e5f6-a7b8-9012-cdef-123456789012 \
  --header 'Authorization: Bearer <token>'
```

</RequestExample>

<ResponseExample>

```json 204

```

```json 404
{
  "statusCode": 404,
  "message": "Professor not found",
  "error": "Not Found"
}
```

</ResponseExample>
